# RecruitOS Prototype

A lightweight recruiting CRM prototype built as a single-page browser app.

## What's Included

- Dashboard analytics
- Candidate CRM
- Client CRM
- Pipeline board
- Global search
- Candidate intake form
- Login/authentication mockup
- Mobile-responsive layout
- Starter PostgreSQL database schema

## Open the Prototype

Open this file in a browser:

```text
outputs/codex-can-build-immediately.html
```

No installation is required for the current prototype.

## Database Schema

The starter PostgreSQL schema is here:

```text
outputs/recruitos-schema.sql
```

It includes tables for users, clients, candidates, searches, pipeline items, and notes.

## Suggested Next Steps

1. Move the prototype into a real app framework such as Next.js, React, or Vue.
2. Connect authentication to a service such as Supabase, Clerk, or Auth0.
3. Connect the SQL schema to PostgreSQL or Supabase.
4. Replace the sample records with live database data.
5. Add user roles, saved searches, notes, file uploads, and client portals.
